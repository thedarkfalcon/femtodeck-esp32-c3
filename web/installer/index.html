Femto OS for C3 Headless

No-screen ESP32-C3 firmware with a tiny LED/button launcher for headless utilities.

  • Chip: ESP32-C3
  • Screen: none
  • Fresh boards start in the LED menu.
  • Includes Mouse Emulator and Distributed Miner slave modes.
  • Double-tap in an app to save autolaunch; double-tap in the menu clears it.
  • Triple-tap inside an app clears that app's pairing.

What is Femto OS?

Femto OS is a tiny ESP32 app launcher with board-specific interfaces and shared app logic. The C3 build is designed around extreme constraints, while the T-Display build uses the larger color screen for dashboards and smoother two-button controls.

Before flashing

Use a data-capable USB cable, close any serial monitor using the port, and hold BOOT while connecting the board if it will not enter flashing mode.
